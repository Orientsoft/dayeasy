我们以一道综合题为例子来说明如何标记：

这是综合题的题干部分，综合题的题干部分，综合题的题干部分，综合题的题干部分，综合题的题干部分，综合题的题干部分 【题干】
(1).第一小问的描述（这里是一个主观题）【小问】
(2).第二小问的描述（这里是一个客观题）【小问】
A.这里是选项内容
B.这里是选项内容
C.这里是选项内容
D.这里是选项内容【选项】

四种标签

【题干】

标识题干部分内容，在一个题中仅且只有一个该标签；如果打上该标签，从开始到第一次出现该标签的位置，视为题干内容。

【小问】

标识小问部分内容。
标记位置：可在每个小问结束的时候标记上
（1）如果小问与小问之间有换行或者4个空格以上的距离则可以省略该标记；
（2）如果没有其他标记，则最后一个小问的标记可以省略，否则最后最后一个不能省略。

【选项】

标识客观题选项部分内容。
标记位置：该标识如果有“【小问】”标记，则出现在”【小问】”标记的后面的选项结束位置，选项与选项之间必须以换行或者4个以上的空格隔开。如果没有”【小问】”标记，则出现在“【题干】”后面选项结束位置，如果有多组选项，选项与选项直接以4个以上的空格隔开，选项组与选项组之间以换行隔开，则只标识最后一个选项组“标记”。
注：该标记不能与“【答案】”标记一起使用。

【答案】

标识主观题答案的标记。
标记位置：该标记用来标识主观题答案部分内容，一个答案一个标记对应一个小问，如果答案与答案之间一换行或者4个以上空格隔开，则可以省略该标记，如果多个小问最后一个标记则不能省略，如果没有小问则最后一个可以省略。
注：该标记不能与“【选项】”标记一起使用。
